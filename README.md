# Image Upload & Gallery Application (PHP)

## Overview
This project allows users to upload images and view them in a gallery.

## Features
- Image upload using form
- File validation (type & size)
- Session messages
- Dynamic gallery display

## Validation Rules
- Allowed types: JPG, JPEG, PNG, GIF
- Max size: 2MB

## Technologies Used
- PHP
- HTML
- CSS

## How to Run
1. Install XAMPP/WAMP
2. Place project in htdocs
3. Start Apache server
4. Open:
   http://localhost/image-gallery-app
